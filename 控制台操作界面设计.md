# 控制台操作界面设计

## 1. 系统概述

控制台操作界面是macOS本地化智能体编码工具的核心组件之一，提供命令行交互能力，支持用户通过终端执行各种操作，包括智能体控制、VS Code集成、本地系统管理和插件操作等。

## 2. 功能需求

### 2.1 核心功能
- 命令解析与执行
- 命令历史记录与自动补全
- 输出格式化与着色
- 多命令支持与管道操作
- 命令执行状态管理
- 帮助系统与命令文档

### 2.2 高级功能
- 智能命令建议
- 命令执行环境管理
- 自定义命令与别名
- 批量命令执行
- 命令执行结果导出

### 3.2 核心组件

#### 3.2.1 命令解析器
- **功能**：解析用户输入的命令，识别命令名称、参数和选项
- **实现方案**：
  - 使用命令行参数解析库（如commander.js）
  - 支持命令嵌套和子命令
  - 实现命令参数验证
  - 提供命令帮助信息

#### 3.2.2 命令执行器
- **功能**：执行解析后的命令，调用相应的功能模块
- **实现方案**：
  - 实现命令执行上下文
  - 支持同步和异步命令执行
  - 处理命令执行错误
  - 管理命令执行状态

#### 3.2.3 输出格式化器
- **功能**：格式化命令执行结果，提供友好的输出
- **实现方案**：
  - 支持多种输出格式（文本、JSON、表格等）
  - 实现输出着色
  - 处理不同类型的输出（成功、错误、警告等）
  - 支持分页输出

#### 3.2.4 历史记录管理器
- **功能**：管理命令历史记录，支持历史命令查询和重用
- **实现方案**：
  - 本地存储命令历史
  - 支持历史命令搜索
  - 实现命令自动补全
  - 提供历史命令统计

  // 解析命令输入

  name: string;
  description: string;
  arguments: Array<{
    name: string;
    type: string;
    required: boolean;
    description: string;
  }>;
  options: Array<{
    name: string;
    short: string;
    type: string;
    default: any;
    description: string;
  }>;
  handler: (args: any, options: any) => Promise<any>;
  subcommands?: Map<string, CommandDefinition>;
}

interface Command {
  name: string;
  arguments: any[];
  options: Record<string, any>;
  subcommand?: Command;
}

## 6. 技术实现方案

### 6.1 控制台启动流程

```typescript
// consoleInterface.ts
import { CommandParser } from './commandParser';
import { CommandExecutor } from './commandExecutor';
import { OutputFormatter } from './outputFormatter';
import { HistoryManager } from './historyManager';
import * as readline from 'readline';

export class ConsoleInterface {
  private rl: readline.Interface;
  private commandParser: CommandParser;
  private commandExecutor: CommandExecutor;
  private outputFormatter: OutputFormatter;
  private historyManager: HistoryManager;

  constructor() {
    this.commandParser = new CommandParser();
    this.commandExecutor = new CommandExecutor();
    this.outputFormatter = new OutputFormatter();
    this.historyManager = new HistoryManager();
    
    // 初始化readline
    this.rl = readline.createInterface({
      input: process.stdin,
      output: process.stdout,
      prompt: 'smart-agent> ',
      historySize: 1000,
      completer: this.completer.bind(this)
    });
    
    // 注册默认命令
    this.registerDefaultCommands();
  }

  start(): void {
    console.log('欢迎使用Smart Agent Console');
    console.log('输入 help 查看可用命令');
    console.log('');
    
    this.rl.prompt();
    
    this.rl.on('line', async (input) => {
      const trimmedInput = input.trim();
      if (trimmedInput) {
        // 添加到历史记录
        this.historyManager.addCommand(trimmedInput);
        
        try {
          // 解析命令
          const command = this.commandParser.parse(trimmedInput);
          
          // 执行命令
          const result = await this.commandExecutor.executeCommand(command);
          
          // 格式化输出
          const output = this.outputFormatter.formatOutput(result);
          console.log(output);
        } catch (error) {
          console.error(this.outputFormatter.formatError((error as Error).message));
        }
      }
      
      this.rl.prompt();
    });
    
    this.rl.on('close', () => {
      console.log('再见!');
      process.exit(0);
    });
  }

  private completer(line: string): [string[], string] {
    // 实现命令自动补全
    const commands = this.commandParser.getCommands().map(cmd => cmd.name);
    const hits = commands.filter(cmd => cmd.startsWith(line));
    return [hits.length ? hits : commands, line];
  }

  private registerDefaultCommands(): void {
    // 注册核心命令
    this.commandParser.registerCommand('agent', {
      name: 'agent',
      description: '智能体控制',
      arguments: [{ name: 'action', type: 'string', required: true, description: '操作类型' }],
      options: [{ name: 'verbose', short: 'v', type: 'boolean', default: false, description: '详细输出' }],
      handler: async (args, options) => {
        // 实现智能体控制逻辑
        return { success: true, output: `执行智能体操作: ${args.action}` };
      }
    });
    
    // 注册其他命令...
  }
}
```

function formatTableOutput(data: any[]): string {
  const table = require('table');
  const chalk = require('chalk');
  
  // 准备表格数据
  const headers = Object.keys(data[0]).map(key => chalk.bold(key));
