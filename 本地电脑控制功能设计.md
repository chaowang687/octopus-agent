# 本地电脑控制功能设计

## 1. 系统概述

本地电脑控制功能是macOS本地化智能体编码工具的核心组件之一，负责管理和控制本地电脑的各种操作，包括打开网页、文件操作、应用控制、系统状态监控等，为用户提供便捷的系统管理能力。

## 2. 功能需求

### 2.1 核心功能
- 网页打开与管理
- 文件操作（创建、复制、移动、删除等）
- 应用程序控制（启动、停止、切换等）
- 系统状态监控（CPU、内存、磁盘等）
- 网络操作（下载、上传、网络状态等）

### 2.2 高级功能
- 自动化任务调度
- 系统配置管理
- 剪贴板操作
- 屏幕捕获与录制
- 系统事件监听

#### 3.2.1 网页控制器
- **功能**：打开和管理网页浏览器
- **实现方案**：
  - 使用`child_process`执行`open`命令
  - 支持指定浏览器
  - 实现网页书签管理
  - 支持多标签页控制

#### 3.2.2 文件系统操作
- **功能**：执行文件和目录操作
- **实现方案**：
  - 使用Node.js的`fs`模块
  - 支持文件读写、复制、移动、删除
  - 实现目录遍历和搜索
  - 支持文件权限管理

#### 3.2.3 应用控制器
- **功能**：控制本地应用程序
- **实现方案**：
  - 使用AppleScript执行应用操作
  - 支持应用启动、停止、切换
  - 实现应用窗口管理
  - 支持应用参数配置

#### 3.2.4 系统监控
- **功能**：监控系统状态和资源使用
- **实现方案**：
  - 使用`child_process`执行系统命令
  - 实现CPU、内存、磁盘监控
  - 支持网络状态监控
  - 提供系统事件通知

### 5.1 网页控制实现

```typescript
// WebController.ts
import { exec } from 'child_process';
import { promisify } from 'util';

const execAsync = promisify(exec);

export class WebController {
  async open(url: string, options?: WebOptions): Promise<void> {
    let command = 'open';
    
    // 处理浏览器选项
    if (options?.browser) {
      command += ` -a "${options.browser}"`;
    }
    
    // 处理新窗口选项
    if (options?.newWindow) {
      command += ' -n';
    }
    
    // 添加URL
    command += ` "${url}"`;
    
    // 执行命令
    await execAsync(command);
  }

  async openMultiple(urls: string[]): Promise<void> {
    for (const url of urls) {
      await this.open(url);
    }
  }

  async close(browser?: string): Promise<void> {
    const command = browser 
      ? `pkill -f "${browser}"` 
      : 'pkill -f "Safari|Chrome|Firefox"';
    
    await execAsync(command).catch(() => {
      // 忽略错误，可能是因为浏览器未运行
    });
  }

  async getBrowserStatus(browser: string = 'Safari'): Promise<BrowserStatus> {
    try {
      // 使用AppleScript获取浏览器状态
      const script = `tell application "${browser}" to get {name, version, count of windows}`;
      const { stdout } = await execAsync(`osascript -e '${script}'`);
      
      const parts = stdout.trim().split(', ');
      return {
        name: parts[0],
        version: parts[1],
        windows: parseInt(parts[2]),
        tabs: 0, // 需要额外脚本获取
        active: true
      };
    } catch (error) {
      return {
        name: browser,
        version: 'unknown',
        windows: 0,
        tabs: 0,
        active: false
      };
    }
  }
}

interface WebOptions {
  browser?: string;
  newWindow?: boolean;
  private?: boolean;
}

interface BrowserStatus {
  name: string;
  version: string;
  windows: number;
  tabs: number;
  active: boolean;
}
```

### 5.2 文件系统操作实现

```typescript
// FileSystem.ts
import * as fs from 'fs-extra';
import * as path from 'path';

export class FileSystem {
  async createFile(filePath: string, content: string = ''): Promise<void> {
    // 确保目录存在
    const dir = path.dirname(filePath);
    await fs.ensureDir(dir);
    
    // 创建文件
    await fs.writeFile(filePath, content);
  }

  async readFile(filePath: string): Promise<string> {
    return await fs.readFile(filePath, 'utf8');
  }

  async writeFile(filePath: string, content: string): Promise<void> {
    // 确保目录存在
    const dir = path.dirname(filePath);
    await fs.ensureDir(dir);
    
    // 写入文件
    await fs.writeFile(filePath, content);
  }

  async copyFile(source: string, destination: string): Promise<void> {
    await fs.copy(source, destination);
  }

  async moveFile(source: string, destination: string): Promise<void> {
    await fs.move(source, destination);
  }

  async deleteFile(filePath: string): Promise<void> {
    await fs.remove(filePath);
  }

  async listDirectory(dirPath: string): Promise<FileInfo[]> {
    const files = await fs.readdir(dirPath, { withFileTypes: true });
    
    const fileInfos: FileInfo[] = [];
    for (const file of files) {
      const fullPath = path.join(dirPath, file.name);
      const stats = await fs.stat(fullPath);
      
      fileInfos.push({
        name: file.name,
        path: fullPath,
        size: stats.size,
        type: file.isDirectory() ? 'directory' : 'file',
        modified: stats.mtimeMs,
        accessed: stats.atimeMs,
        permissions: stats.mode.toString(8)
      });
    }
    
    return fileInfos;
  }

  async searchFiles(pattern: string, directory: string = process.cwd()): Promise<string[]> {
    const files: string[] = [];
    
    async function search(dir: string) {
      const entries = await fs.readdir(dir, { withFileTypes: true });
      
      for (const entry of entries) {
        const fullPath = path.join(dir, entry.name);
        
        if (entry.isDirectory()) {
          await search(fullPath);
        } else if (entry.name.includes(pattern)) {
          files.push(fullPath);
        }
      }
    }
    
    await search(directory);
    return files;
  }
}

interface FileInfo {
  name: string;
  path: string;
  size: number;
  type: 'file' | 'directory';
  modified: number;
  accessed: number;
  permissions: string;
}
```

### 5.3 应用控制实现

```typescript
// AppController.ts
import { exec } from 'child_process';
import { promisify } from 'util';

const execAsync = promisify(exec);

export class AppController {
  async start(appName: string, options?: AppOptions): Promise<void> {
    let command = `open -a "${appName}"`;
    
    // 处理启动参数
    if (options?.args && options.args.length > 0) {
      command += ` -- ${options.args.join(' ')}`;
    }
    
    // 执行命令
    await execAsync(command);
  }

  async stop(appName: string): Promise<void> {
    const script = `tell application "${appName}" to quit`;
    await execAsync(`osascript -e '${script}'`);
  }

  async restart(appName: string): Promise<void> {
    await this.stop(appName);
    await new Promise(resolve => setTimeout(resolve, 1000)); // 等待一秒
    await this.start(appName);
  }

  async switchTo(appName: string): Promise<void> {
    const script = `tell application "${appName}" to activate`;
    await execAsync(`osascript -e '${script}'`);
  }

  async getAppStatus(appName: string): Promise<AppStatus> {
    try {
      // 检查应用是否运行
      const { stdout } = await execAsync(`pgrep -l "${appName}"`);
      const lines = stdout.trim().split('\n');
      
      if (lines.length > 0 && lines[0] !== '') {
        const [pid, name] = lines[0].split(' ');
        return {
          name: appName,
          pid: parseInt(pid),
          status: 'running',
          windows: 0, // 需要AppleScript获取
          cpu: 0, // 需要额外命令获取
          memory: 0 // 需要额外命令获取
        };
      }
    } catch (error) {
      // 应用未运行
    }
    
    return {
      name: appName,
      pid: 0,
      status: 'stopped',
      windows: 0,
      cpu: 0,
      memory: 0
    };
  }

  async getRunningApps(): Promise<AppStatus[]> {
    const { stdout } = await execAsync('ps aux | grep -E "^[^ ]+ .+ /Applications/" | grep -v grep');
    const lines = stdout.trim().split('\n');
    
    const apps: AppStatus[] = [];
    for (const line of lines) {
      if (line) {
        const parts = line.split(/\s+/);
        const pid = parseInt(parts[1]);
        const appPath = parts[parts.length - 1];
        const appName = appPath.split('/').pop()?.replace('.app', '') || 'Unknown';
        
        apps.push({
          name: appName,
          pid,
          status: 'running',
          windows: 0,
          cpu: parseFloat(parts[2]),
          memory: parseFloat(parts[3])
        });
      }
    }
    
    return apps;
  }
}

interface AppOptions {
  args?: string[];
  environment?: Record<string, string>;
  directory?: string;
}

interface AppStatus {
  name: string;
  pid: number;
  status: 'running' | 'stopped';
  windows: number;
  cpu: number;
  memory: number;
}
```

### 5.4 系统监控实现

```typescript
// SystemMonitor.ts
import { exec } from 'child_process';
import { promisify } from 'util';
import si from 'systeminformation';

const execAsync = promisify(exec);

export class SystemMonitor {
  private monitoringInterval: NodeJS.Timeout | null = null;

  async getCpuInfo(): Promise<CpuInfo> {
    const cpu = await si.cpu();
    const currentLoad = await si.currentLoad();
    
    return {
      model: cpu.brand,
      cores: cpu.cores,
      speed: cpu.speed,
      usage: currentLoad.currentLoad
    };
  }

  async getMemoryInfo(): Promise<MemoryInfo> {
    const mem = await si.mem();
    return {
      total: mem.total,
      free: mem.free,
      used: mem.used,
      usage: mem.used / mem.total * 100
    };
  }

  async getDiskInfo(): Promise<DiskInfo[]> {
    const disks = await si.fsSize();
    return disks.map(disk => ({
      device: disk.fs,
      mount: disk.mount,
      type: disk.type,
      total: disk.size,
      free: disk.size - disk.used,
      used: disk.used,
      usage: disk.use
    }));
  }

  async getNetworkInfo(): Promise<NetworkInfo> {
    const networkInterfaces = await si.networkInterfaces();
    const networkStats = await si.networkStats();
    
    return {
      interfaces: networkInterfaces.map(intf => ({
        name: intf.iface,
        ip4: intf.ip4,
        ip6: intf.ip6,
        mac: intf.mac,
        status: intf.internal ? 'down' : 'up'
      })),
      status: networkStats.length > 0 ? 'online' : 'offline',
      ipAddress: networkInterfaces.find(intf => !intf.internal)?.ip4 || 'N/A',
      macAddress: networkInterfaces.find(intf => !intf.internal)?.mac || 'N/A'
    };
  }

  async getSystemLoad(): Promise<SystemLoad> {
    const load = await si.currentLoad();
    return {
      1min: load.load1,
      5min: load.load5,
      15min: load.load15
    };
  }

  async getSystemStatus(): Promise<SystemStatus> {
    return {
      cpu: await this.getCpuInfo(),
      memory: await this.getMemoryInfo(),
      disk: await this.getDiskInfo(),
      network: await this.getNetworkInfo(),
      load: await this.getSystemLoad(),
      uptime: await this.getUptime()
    };
  }

  monitorSystem(callback: (status: SystemStatus) => void, interval: number = 5000): void {
    this.stopMonitoring(); // 停止之前的监控
    
    this.monitoringInterval = setInterval(async () => {
      try {
        const status = await this.getSystemStatus();
        callback(status);
      } catch (error) {
        console.error('Error monitoring system:', error);
      }
    }, interval);
  }

  stopMonitoring(): void {
    if (this.monitoringInterval) {
      clearInterval(this.monitoringInterval);
      this.monitoringInterval = null;
    }
  }

  async getUptime(): Promise<number> {
    const uptime = await si.time();
    return uptime.uptime;
  }

  async getSystemVersion(): Promise<SystemVersion> {
    const os = await si.osInfo();
    return {
      os: os.distro,
      version: os.release,
      build: os.build,
      arch: os.arch
    };
  }
}

  model: string;
  cores: number;
  speed: number;
  usage: number;
}

  total: number;
  free: number;
  used: number;
  usage: number;

  total: number;

  usage: number;
}

  macAddress: string;
}
```

## 6. 集成方案

### 6.1 与智能体引擎集成

#### 6.1.1 智能体系统控制

- 智能体可以通过本地系统控制器执行系统操作
- 支持智能体自动化任务调度
- 提供系统状态作为智能体决策依据

#### 6.1.2 示例场景

```typescript
// 智能体执行系统任务示例
async function executeSystemTask(task: string): Promise<void> {
  if (task.includes('打开网页')) {
    // 提取URL
    const urlMatch = task.match(/打开网页\s+(https?:\/\/[^\s]+)/);
    if (urlMatch) {
      const url = urlMatch[1];
      await this.localSystemController.openWeb(url);
      return `已打开网页: ${url}`;
    }
  } else if (task.includes('创建文件')) {
    // 提取文件名和内容
    const fileMatch = task.match(/创建文件\s+(\S+)\s+内容\s+(.*)/);
    if (fileMatch) {
      const [, filePath, content] = fileMatch;
      await this.localSystemController.fileOperation('create', filePath, content);
      return `已创建文件: ${filePath}`;
    }
  }
  
  return '任务解析失败，请提供更明确的指令';
```

#### 6.3.2 插件集成示例

```typescript
// 插件系统集成示例
class SystemMonitorPlugin {
  private systemController: LocalSystemController;
  private monitoringInterval: NodeJS.Timeout | null = null;

  constructor(systemController: LocalSystemController) {
    this.systemController = systemController;
  }

  activate() {
    // 启动系统监控
    this.monitoringInterval = setInterval(async () => {
      const status = await this.systemController.getSystemStatus();
      console.log('System Status:', status);
      
      // 检查系统负载
      if (status.cpu.usage > 80) {
        console.warn('CPU usage is high:', status.cpu.usage);
      }
      
      if (status.memory.usage > 80) {
        console.warn('Memory usage is high:', status.memory.usage);
      }
    }, 10000);
  }

  deactivate() {
    if (this.monitoringInterval) {
      clearInterval(this.monitoringInterval);
    }
  }
}
```
