# 智能管家项目问题解决功能测试指南

## 功能概述

智能管家已经成功集成到项目管理页面，具备以下核心能力：

### 1. 项目记忆系统
- **记忆类型**：问题（issue）、解决方案（solution）、偏好（preference）、模式（pattern）、命令（command）
- **记忆存储**：持久化存储在 `~/.trae-ai/project-memories.json`
- **智能搜索**：根据关键词搜索相关记忆，按重要性和访问频率排序
- **自动清理**：定期清理旧记忆，保留重要和频繁访问的记忆

### 2. 项目管理工具
智能管家内置了6个项目管理工具：

| 工具名称 | 类别 | 功能描述 |
|---------|------|---------|
| install_dependencies | setup | 安装项目依赖（npm install） |
| start_dev_server | development | 启动开发服务器 |
| build_project | deployment | 构建项目 |
| run_tests | development | 运行测试 |
| clean_node_modules | maintenance | 清理node_modules并重新安装 |
| check_project_health | maintenance | 检查项目健康状态 |

### 3. 智能问题解决
智能管家能够：
- 自动识别问题类型（依赖、启动、构建等）
- 搜索历史记忆中的解决方案
- 自动执行相应的项目管理工具
- 记录问题和解决方案到记忆系统
- 提供详细的解决步骤

## 测试步骤

### 测试1：依赖问题自动解决

1. **打开项目管理页面**
   - 在应用中点击"项目管理"图标
   - 切换到"文件系统项目"标签

2. **选择一个项目**
   - 点击任意一个文件系统项目
   - 查看项目详情

3. **描述依赖问题**
   - 在"智能管家 - 问题解决"区域输入：
     ```
     项目依赖没有安装，npm install失败
     ```

4. **点击"让智能管家解决"**
   - 智能管家会自动识别这是依赖问题
   - 自动执行 `install_dependencies` 工具
   - 显示解决方案和建议步骤

5. **验证结果**
   - 检查是否显示"✓ 解决方案"
   - 查看建议步骤是否正确
   - 检查项目记忆是否已保存

### 测试2：启动问题自动解决

1. **描述启动问题**
   - 在"智能管家 - 问题解决"区域输入：
     ```
     项目无法启动，开发服务器启动失败
     ```

2. **点击"让智能管家解决"**
   - 智能管家会自动识别这是启动问题
   - 自动执行 `start_dev_server` 工具
   - 显示解决方案和建议步骤

3. **验证结果**
   - 检查是否显示"✓ 解决方案"
   - 查看建议步骤是否正确
   - 检查开发服务器是否成功启动

### 测试3：构建问题自动解决

1. **描述构建问题**
   - 在"智能管家 - 问题解决"区域输入：
     ```
     项目构建失败，build命令报错
     ```

2. **点击"让智能管家解决"**
   - 智能管家会自动识别这是构建问题
   - 自动执行 `build_project` 工具
   - 显示解决方案和建议步骤

3. **验证结果**
   - 检查是否显示"✓ 解决方案"
   - 查看建议步骤是否正确

### 测试4：项目健康检查

1. **描述一般问题**
   - 在"智能管家 - 问题解决"区域输入：
     ```
     项目运行不正常，不知道哪里出了问题
     ```

2. **点击"让智能管家解决"**
   - 智能管家会自动执行 `check_project_health` 工具
   - 显示项目健康状态检查结果
   - 提供建议步骤

3. **验证结果**
   - 检查是否显示健康检查结果
   - 查看建议步骤是否合理

### 测试5：记忆系统验证

1. **多次解决相同问题**
   - 重复测试1-4中的任意一个问题
   - 观察智能管家是否更快找到解决方案

2. **检查记忆文件**
   - 打开终端，执行：
     ```bash
     cat ~/.trae-ai/project-memories.json
     ```
   - 查看是否保存了问题和解决方案记忆

3. **搜索记忆**
   - 在"智能管家 - 问题解决"区域输入之前解决过的问题关键词
   - 点击"让智能管家解决"
   - 验证是否从记忆中找到解决方案

### 测试6：未知问题处理

1. **描述未知问题**
   - 在"智能管家 - 问题解决"区域输入：
     ```
     项目有一个奇怪的bug，不知道怎么解决
     ```

2. **点击"让智能管家解决"**
   - 智能管家无法自动解决
   - 会显示"✗ 分析结果"
   - 提供通用的建议步骤

3. **验证结果**
   - 检查是否显示合理的建议步骤
   - 验证问题是否被记录到记忆系统

## 调试方法

### 查看智能管家日志
1. 打开应用的开发者工具（Cmd+Option+I）
2. 切换到Console标签
3. 查看以 `[SmartButler]` 开头的日志

### 查看项目记忆文件
```bash
cat ~/.trae-ai/project-memories.json
```

### 查看项目信息文件
```bash
cat ~/.trae-ai/projects.json
```

### 手动测试IPC接口
在开发者工具的Console中执行：
```javascript
// 获取所有项目记忆
window.electron.butler.getProjectMemories('project-id').then(console.log)

// 搜索记忆
window.electron.butler.searchProjectMemories('依赖').then(console.log)

// 获取项目管理工具
window.electron.butler.getProjectTools().then(console.log)

// 执行工具
window.electron.butler.executeProjectTool('check_project_health', '/path/to/project').then(console.log)
```
