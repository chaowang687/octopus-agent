```
function reasoning(input):
    // 初始化
    state = initialize(input)
    
    // 推理循环
    while not converged(state):
        // 思考
        thought = think(state)
        
        // 行动
        action = decide(thought)
        
        // 观察
        observation = observe(action)
        
        // 更新状态
        state = update(state, observation)
    
    // 返回结果
    return result(state)
```
