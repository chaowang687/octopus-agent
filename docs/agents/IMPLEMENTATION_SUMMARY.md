## 使用指南

### 创建新智能体

1. 复制模板：
   ```bash
   cp docs/agents/templates/agent-template.md docs/agents/core/my-agent.md
   ```

2. 创建配置文件：
   ```bash
   cp config/agents/project-manager.json config/agents/my-agent.json
   ```

3. 填写文档内容
4. 实现智能体逻辑
5. 编写测试
6. 更新README

### 查看智能体文档

- 查看所有智能体：[README.md](README.md)
- 查看特定智能体：进入对应目录
- 学习开发：阅读[开发指南](guides/agent-development-guide.md)
