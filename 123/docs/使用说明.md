# 使用说明文档

## 项目介绍

简易笔记本是一个基于浏览器的在线笔记应用，所有数据存储在您的浏览器本地，无需注册登录，打开即可使用，简洁高效，保护隐私。

## 功能说明

### 基础功能

- **新建笔记**: 点击右上角「新建笔记」按钮创建新笔记
- **选择笔记**: 在左侧/上方笔记列表点击任意笔记即可编辑
- **编辑笔记**: 在编辑区域直接输入文字内容
- **保存笔记**: 点击底部「保存」按钮保存修改
- **删除笔记**: 点击底部「删除」按钮，确认后删除当前笔记

### 特色特性

- 🔒 **隐私保护**: 所有数据都存在您的浏览器本地，不会上传到任何服务器
- 📱 **响应式设计**: 完美适配电脑和手机，随时随地方便记录
- ⚡ **极速打开**: 无需加载，打开即可使用，流畅不卡顿
- 💾 **自动持久化**: 刷新页面数据不会丢失，永久保存

## 安装和运行

### 直接打开（最简单）

1. 下载项目所有文件到本地
2. 找到 `src/index.html` 文件
3. 用浏览器直接打开这个文件即可使用

### 使用本地服务器（推荐）

如果您有Node.js环境，可以启动一个本地开发服务器：

```bash
# 进入项目目录
cd /path/to/123/src

# 使用npx启动静态服务器
npx serve .

# 或者使用python
python3 -m http.server 8000
```

然后在浏览器打开 `http://localhost:8000` 即可使用。

### 部署到云端

项目是纯静态文件，可以部署到任何静态文件托管服务：

- **GitHub Pages**: 直接推送代码到GitHub，开启Pages即可
- **Vercel**: 导入项目，一键部署
- **Netlify**: 同样支持一键部署
- **你的服务器**: 上传静态文件到任意目录即可

## 浏览器兼容性

支持所有现代浏览器：

- Chrome/Edge (推荐)
- Firefox
- Safari
- 其他基于Chromium的浏览器

## 数据管理

### 数据存储位置

所有笔记都存储在您浏览器的LocalStorage中，只有您能访问这些数据。

### 数据备份

目前版本如果需要备份，可以手动复制笔记内容保存到其他地方。未来版本会增加导入导出功能。

### 数据清除

清除浏览器缓存会删除所有笔记，请谨慎操作。

## 使用技巧

1. **快速开始**: 打开页面后直接点击「新建笔记」即可开始记录
2. **自动标题**: 笔记标题自动使用第一行内容，方便识别
3. **时间记录**: 每个笔记会自动记录创建和修改时间
4. **大笔记支持**: 只要浏览器LocalStorage容量足够，可以存储大笔记

## 常见问题

**Q: 刷新页面后笔记不见了怎么办？**
A: 请检查您是否启用了浏览器隐私模式，或者清除了浏览器缓存。隐私模式下LocalStorage在关闭后会被自动清除。

**Q: 提示内容过大无法保存怎么办？**
A: LocalStorage一般有5MB的总容量限制，请清理一些不需要的旧笔记后再试。

**Q: 可以同步到其他设备吗？**
A: 当前版本不支持跨设备同步，因为所有数据都存在本地。未来版本可能会增加云同步功能。

**Q: 支持富文本编辑吗？**
A: 当前版本只支持纯文本编辑，这是设计上的选择，保持极简。如果需要富文本可以后续扩展。

## 技术支持

如果遇到问题，可以检查：
1. 确认使用的是现代浏览器
2. 确认JavaScript没有被禁用
3. 确认浏览器支持LocalStorage
